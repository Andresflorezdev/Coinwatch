## ✨ Features

- 📊 **Live price chart** — interactive 30-point history per coin
- 💹 **Real-time ticker** — prices update every 1.8 s with simulated market movement
- 🏦 **Portfolio summary** — total value and average daily change at a glance
- 🎨 **Dark glassmorphism UI** — Space Mono + Syne typography, glow orbs, pulse animations
- 📱 **Responsive** — adapts from desktop to mobile

## 🚀 Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run serve
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

## 🛠️ Scripts

| Command | Description |
|---------|-------------|
| `npm run serve` | Start dev server with hot-reload |
| `npm run build` | Build for production (`dist/`) |
| `npm run lint` | Lint & auto-fix files |

## ⚙️ CI

Every push and pull request to `main` automatically runs **lint** and **build** via GitHub Actions.
